Asistencia para Clientes y Soporte Técnico

Opciones de soporte en línea

  • La Base de Conocimientos es la nueva base de datos de soporte de Apple que contiene información técnica actualizada y reciente sobre productos, descargas de software, consejos para resolver problemas y artículos de apoyo.
  • Los Foros de Discusión facilitan el intercambio de ideas, problemas y soluciones. Usa los foros de discusión sobre productos específicos para publicar tus preguntas y recibir respuestas.
  • Las Descargas de Software ahora forman parte de la Base de Conocimientos. También puedes encontrar descargas destacadas y recientes con enlaces a otras descargas archivadas.
