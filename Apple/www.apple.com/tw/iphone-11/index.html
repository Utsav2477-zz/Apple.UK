全新雙相機系統。

從廣角到超廣角，任你拍攝。重新設計的介面運用全新超廣角相機，為你顯示取景框外的畫面，且一併捕捉。拍攝和剪輯影片，處理起來也和照片一樣容易。這部全球最受歡迎的相機，現在，視角煥然一新。

與 iPhone 11 Pro 相機系統比較

iPhone 11 Pro 配備相同的廣角與超廣角相機，並有全新的望遠相機。三相機系統搭配運作，可帶來四倍光學變焦範圍，提供更多的取景選項。

提升影片新境界，
一絲一毫不放過。

以智慧型手機中最高的影片畫質進行拍攝與剪輯。 iPhone 11 的每部相機都能以 60 fps 拍攝精緻清晰的 4K 影片。超廣角相機可捕捉四倍寬廣的取景範圍，因此非常適合拍攝愛犬接飛盤這類的動作鏡頭。如果你拍的是孩子的鋼琴演奏這類表演活動，當你拉近畫面時，聲音也會跟著拉近。而且，剪輯影片現在就和編輯照片一樣容易。

全新超廣角相機。

大視野，現正當道。全新超廣角相機可捕捉四倍寬廣的取景範圍。想想那些綿延不絕的動人風光、整個舞台與音樂節萬頭攢動的盛況，或是當全家遠赴千里度假時，在名勝風景中來張全家福合照。

夜間模式新登場。

在低光源環境拍攝，未必要拍出暗不見光的照片。「夜間」模式是全新功能，可自動開啟，讓你在低光源環境下，拍攝前所未見的精彩美照。不用閃光燈，色彩更自然，而且照片也更明亮。

出色夜景照片背後的驅動力。

在低光源環境拍出精彩照片，向來都是攝影上的挑戰。快門時間變長，可能會造成畫面模糊；保有暗部細節的同時，不讓亮部過度曝光；同時還要確保色彩看起來自然舒適。

有了 iPhone 11 與 iPhone 11 Pro，全新的廣角相機感光元件能與智慧軟體和 A13 仿生搭配運作，讓你以 iPhone 成就以往意想不到的可能：在微弱光源的環境下，拍出亮麗動人、細節豐富的照片。

「夜間」模式會在需要時自動開啟，像是在燭光照明的餐廳裡。輕點快門時，相機會拍攝多張照片，而光學影像穩定功能則會穩定鏡頭。

接著相機軟體上場，它會對齊影像、修正晃動；捨棄過於模糊的部分，補上更清晰的細節；調整對比度，平衡影像；微調色彩，讓畫面看起來舒適自然；然後再以智慧方式消除雜點並增強細節，進而產生最終影像。

繁複的微調造就卓然出眾的夜景照片，細節更豐富、雜點更微小，且具有真實的時間感與空間感。

人像，人人上相。

為人像帶來新風貌。

結合新的人像表現與更多的光線控制方式，iPhone 11 的雙相機能搭配運作．拍出驚豔四座的影像。「人像」模式現在能用來拍攝你所喜愛的一切題材，讓好朋友和毛小孩都上相。

iOS 13 內的「高色調燈光黑白」效果，能讓你的人像照變為精美的攝影棚風格黑白照片。透過「人像光線控制」功能，改變拍攝主體上的光線強度，創造出你想要的照片樣貌。

A13 仿生效能強大，讓事事快速流暢，且以最節能的方式運行，電池續航力得以更長。事實上，先進的 A13 仿生，技術遙遙領先。

拍攝 60 fps 的 4K 影片需要強大處理威力。

為了實現前所未有的速度與先進的機器學習技術，我們致力打造出 A13 仿生。因為它，iPhone 11 才有強大威力，進行需要大量處理器效能的作業。例如在拍攝延伸動態範圍達 60 fps 的 4K 影片時，海量資訊會同時湧進視訊編碼器。對絕大多數晶片而言，處理這些海量資訊可說是一大挑戰，但 A13 卻是輕鬆以對、手到擒來。

你可想像影片就是一連串的單格畫面串接而成，靠著相機感光元件超乎想像的快速，讓 iPhone 11 能以每秒 120 格的畫面，交替產生標準曝光與短時間曝光畫面。

影像訊號處理器與視訊編碼器會立即分析每一畫面，盡可能捕捉鉅細靡遺的細節。神經網路引擎更進一步使用即時機器學習技術，將取景範圍內的不同物件調整至最佳。例如，調整前景人物的光影，同時降低雜點，並增強天空的色彩。一切在轉瞬間自動完成。

以智慧型手機拍攝最高畫質的影片，並非易事，因為有超強大的 A13 仿生，你只覺一切不費吹灰之力。

「你看，根本不用動手。」Face ID，比 Touch ID 更易用、安全。

智慧型手機中最安全的面孔認證技術。

以 Face ID 立即解鎖你的 iPhone，甚至比使用 Touch ID 更安全。只須看一眼，就能登入 app、存取帳號和用 Apple Pay 付款，設定快速又簡單。

iPhone 11 (精心設計，可保護你) 的隱私保護。

iPhone 的每項功能設計，都已將保護個人資訊納入考量。

Face ID 不會儲存或分享你的照片；你所傳送的 iMessage，或是撥打的 FaceTime 通話，資料都會經過加密；「地圖」為你指引方向，但不洩露你行蹤。

對 Face ID 來說，你只是一組數字。這是好事一件。

每項 Apple 產品都以保護你的隱私為念，精心打造。因此，Face ID 不儲存或分享你的照片，改用先進技術，精確測繪你臉部的構成特徵，以確認你的身分。

它是這樣運作的，初次設定 Face ID 時，原深感測相機系統會在你臉上投射 30,000 個不可見的測繪點，再以紅外線鏡頭讀取測繪圖案。A13 仿生會建構你獨特的深度測繪圖，將面孔角度與曲線都轉換為數學表示方式，並成為萬用鑰匙的原型。接著，經過加密，存在你裝置上的安全隔離區並嚴密保護。它不會離開你的 iPhone，也不會備份到 iCloud 或 Apple 伺服器。

每當你解鎖手機或登入 app、使用 Apple Pay 時，Face ID 都會執行同樣程序，投射測繪點，再與你儲存的資料比對。若比對相符，即確認你的身分。Face ID 甚至能適應你的容貌變化，它還能夠感知你的注視，只有在你睜開雙眼直視裝置時，才會為你解鎖 iPhone。

Face ID 因此成為智慧型手機歷來最安全的面孔認證技術。現在，Face ID 更快速、易用，與過去相比，可從更多角度與不同距離辨識。

分享、尋找、玩樂，比以往更精準。
